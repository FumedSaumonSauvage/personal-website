# Personal-website

Source code of my personal website, created with Vite.js.

## How do I run it?

Install React dependencies (incl. Node.js), available on [React JS website](https://react.dev/).

Then, in a bash shell, run:
```bash
cd perso-website
npm install
npm run dev
```


## Notes pour plus tard

Le docker sert potentiellement à rien, c'est pas fonctionnel. Faut juste pull le code sur les serveur, build avec le docker pour creer le dossier client et copier ce dossier sur /var/www du serveur.
